Plot kernel density estimate of tiger activity:


This visualizes the probability distribution of tiger detections across the 24-hour cycle.

This function produces:

* Kernel density curves for each species
* Shaded region representing temporal overlap
* A visual summary of potential interaction or avoidance
